# 传统写法
  1. 绑定事件处理函数 - 数据
     1. 增加项 - 列表数据 -> 增加一项
        {id timestamp, content string, completed: false}
        每一项的视图 -> 列表
     2. 删除项 - 列表数据 -> id -> removeItem
        将对应的视图 -> 列表 -> 删除
     3. 改变完成状态 - 列表数据 -> id -> change completed
        将对应项的完成状态 -> 是否完成 toggle
